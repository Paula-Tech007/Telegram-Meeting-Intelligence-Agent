# Use Cases

## 1. Executive Meeting Summary

A manager sends meeting notes to Telegram and receives a structured summary with decisions, tasks, risks, and next steps.

## 2. Project Follow-up

A project lead captures quick notes after a status meeting. The workflow identifies owners, deadlines, and pending items.

## 3. Sprint Planning Notes

A team sends planning notes and receives a compact list of action items and blockers.

## 4. Customer Success Handoff

A handoff meeting is summarized into decisions, risks, and next actions before the next team takes over.

## 5. Portfolio Demonstration

The project demonstrates:

- n8n workflow orchestration
- Telegram bot automation
- AI-based structured extraction
- Secure credential handling
- Professional GitHub documentation
- Future-ready database design

## Not Recommended Without Additional Controls

- Processing regulated data
- Processing confidential corporate meetings
- Storing raw meeting text without retention policy
- Sending sensitive summaries to shared Telegram groups
- Using production credentials in a public demo environment
